HEGoBaseNode
============

**Inherits:** :ref:`HEGoTrackableNode<class_HEGoTrackableNode>` **<** ``RefCounted``

**Inherited By:** :ref:`HEGoInputReceiverNode<class_HEGoInputReceiverNode>`, :ref:`HEGoTransformableNode<class_HEGoTransformableNode>`

Virtual base class for all HEGo node wrappers.

.. rst-class:: classref-introduction-group

Description
-----------

Stores the underlying Houdini node id and provides shared lifecycle behavior for derived HEGo node classes.



A node id of ``-1`` means no valid Houdini node is currently associated with this instance.



Most derived methods assume :ref:`instantiate()<class_HEGoBaseNode_method_instantiate>` has already produced a valid id.
