HEGoLogManager
==============

**Inherits:** ``Object``

Singleton log signal bridge for HEGo runtime messages.

.. rst-class:: classref-introduction-group

Description
-----------

HEGo logging helpers emit console output and forward messages through this object via :ref:`log_message<class_HEGoLogManager_signal_log_message>`.



Use this class from UI code to display live HEGo logs without parsing console output.



The singleton is created during module initialization and remains available while the extension is loaded.

**log_message**\ (\ message\: ``String``, level\: ``String``\ ) :ref:`đź”—<class_HEGoLogManager_signal_log_message>`

Emitted for each HEGo log entry.



Common level strings are ``MESSAGE``, ``WARNING``, ``ERROR``, and ``LINE``.



The ``message`` payload does not include the console prefix string used by internal print helpers.
