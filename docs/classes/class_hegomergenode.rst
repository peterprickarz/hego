HEGoMergeNode
=============

**Inherits:** :ref:`HEGoInputReceiverNode<class_HEGoInputReceiverNode>` **<** :ref:`HEGoBaseNode<class_HEGoBaseNode>` **<** :ref:`HEGoTrackableNode<class_HEGoTrackableNode>` **<** ``RefCounted``

Wrapper around a Houdini Merge SOP node.

.. rst-class:: classref-introduction-group

Description
-----------

Creates a merge node and connects multiple upstream HEGo nodes into it.



This is the standard way to combine several :ref:`HEGoInputNode<class_HEGoInputNode>` and/or generator outputs before feeding an HDA input.

.. rst-class:: classref-method

|void| **connect_inputs**\ (\ inputs\: ``Array``\ ) :ref:`🔗<class_HEGoMergeNode_method_connect_inputs>`

Connects all nodes in ``inputs`` to this merge node in array order.



Each array item should be an :ref:`HEGoBaseNode<class_HEGoBaseNode>`. Invalid entries are ignored.



Array order maps directly to merge input index order.
