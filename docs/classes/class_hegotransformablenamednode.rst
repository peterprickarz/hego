.. rst-class:: classref-introduction-group

Description
-----------

This class is used as an internal base for named input node types.



The name field is used by C++ implementations during Houdini node creation but is not currently exposed as a bound Godot property or method on this type.



Concrete subclasses may still consume this internal field during :ref:`HEGoBaseNode.instantiate()<class_HEGoBaseNode_method_instantiate>`, so naming can be part of internal node creation even when not script-exposed here.
